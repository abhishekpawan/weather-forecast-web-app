# About The Project

<div align="center">

<img  src="https://github.com/abhishekpawan/weather-forecast-web-app/blob/main/public/img/weatherforecastmockups.png"  alt="Logo"  width="70%">

</div>

It is weather forecast web application, Which give you weather forecast of your desired location of the next 3 days. It's pure node and express based application. The frontend UI is developed using handlebars.

#### You will get weather forcast (Max/Min Temp) for upto 3 days. Including:

- Max and Min Temp

- Chances of Rain

- wind speed

- Humidity

## Getting Started

First, You need to get your `API KEY` from [https://weatherapi.com](https://weatherapi.com) and then add a `.env` file in the source of your project and write the below line of code.

```
WEATHER_API_KEY = (YOUR API KEY)
```

For example, If your API key is : `9784982349837987489223442`
Write:

```
WEATHER_API_KEY = 9784982349837987489223442
```

### Prerequisites

- ### Get a free API Key at [https://weatherapi.com](https://weatherapi.com)

## Installation

### Get a free API Key at [https://weatherapi.com](https://weatherapi.com)

### Clone the repo

```sh

git clone https://github.com/abhishekpawan/weather-forecast-website

```

### To Install all the dependencies

```

npm install

```

### Enter your API in `.env`

```js
WEATHER_API_KEY = yourapikey;
```

### Start API

```

npm start

```
